PokéVerse BD - an e-commerce platform dedicated to anime lovers!
It offers a wide range of anime-themed products including T-shirts, accessories, collectibles, and more.

About the Project:
PokéVerse BD is a modern, full-featured e-commerce website built for a smooth and engaging user experience.

Some key features include:
• Secure authentication system (Firebase)
• Admin dashboard for managing products, orders, users, coupons, etc.
• Shopping cart and checkout system
• Fully responsive for desktop/laptop, tablets, and mobile phone.

Tech Stack: 
React, Tailwind CSS, Node.js, Express, MongoDB, Firebase, Axios, and more.
